# Persiapan
- php versi >= 8.2
- composer
- laragon/xampp

# Installasi
- unduh project atau clone
- buka terminal dan arahkan ke direktori project
- jalankan <code>composer update</code>
- jalankan <code>php artisan key:generate</code>
- buat
- jalankan <code>php artisan key:generate</code>
